# Spring Convention Index

- `layer-and-naming-convention.md`
  DTO, validator, policy, 패키지와 클래스 네이밍 규칙을 정리한다.
- `api-design-convention.md`
  Spring API 설계 시 요청/응답/에러/페이징 규약을 정리한다.
- `flyway-convention.md`
  Flyway migration 디렉터리, 파일명, 버전 전략, 리뷰 체크리스트를 정리한다.

## Rule
