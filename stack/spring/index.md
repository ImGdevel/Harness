# Spring Docs Index

Spring 관련 컨벤션과 스펙 문서의 빠른 탐색용 인덱스다.

## Sections

- `convention/`: Spring 코드 스타일, 계층 규칙, 패키지 규칙
- `spec/`: Spring 아키텍처와 기능 스펙
- `test/`: 테스트 규칙, 픽스처, 슬라이스 테스트 기준
- `entity/`: 엔티티, 매핑, 영속성 관련 규칙
- `snippets/`: 짧게 재사용하는 Spring 코드 예제와 패턴
