# Spring Entity Index

Spring 엔티티와 영속성 관련 문서 인덱스다.

## Documents

- `entity-design-convention.md`
  JPA 엔티티의 생성, 상태 전이, 불변식 유지 규칙을 정리한다.
- `erd-design-convention.md`
  ERD와 물리 스키마 설계 시 따를 네이밍, 키, 관계 규칙을 정리한다.
- `table-definition-writing-guide.md`
  테이블 정의서 작성 시 필수 메타 정보와 컬럼 설명 규칙을 정리한다.

## Rule

- 새 엔티티 또는 매핑 문서를 추가하면 목적과 파일 위치를 기록한다.
- 상위 `../index.md`와 함께 탐색 경로를 유지한다.
