# FastAPI Docs Index

## Sections

- `convention/`: FastAPI 구조, 라우터 규칙, 의존성 주입 규칙
- `spec/`: FastAPI 아키텍처와 기능 스펙
- `test/`: API 테스트와 비동기 테스트 규칙
- `snippets/`: 짧게 재사용하는 FastAPI 코드 예제와 패턴
