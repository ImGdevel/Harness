# Next.js Docs Index

Next.js 관련 컨벤션과 스펙 문서의 빠른 탐색용 인덱스다.

## Sections

- `convention/`: App Router 구조, 서버/클라이언트 경계, 라우팅 규칙
- `spec/`: Next.js 아키텍처와 기능 스펙
- `test/`: 렌더링 검증과 UI 테스트 규칙
- `snippets/`: 짧게 재사용하는 Next.js 코드 예제와 패턴

## Related

- 실제 프로젝트 레지스트리는 `../../project/index.md`와 `../../project/registry.yaml`에서 관리한다.

## Current State

- `convention/`에는 App Router와 서버/클라이언트 경계 기준 문서가 들어가 있다.
- `spec/`, `test/`, `snippets/`는 프로젝트 사례 축적에 맞춰 계속 확장한다.
