# React Docs Index

React 관련 컨벤션과 스펙 문서의 빠른 탐색용 인덱스다.

## Sections

- `convention/`: 컴포넌트 구조, 상태 관리, 폴더 규칙
- `spec/`: 화면 구조와 기능 스펙
- `test/`: UI 테스트, 상호작용 테스트, 렌더링 규칙
- `snippets/`: 짧게 재사용하는 React 코드 예제와 패턴

## Related
