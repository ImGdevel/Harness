# Image Generation Stack Index

이미지 생성 중심 워크스페이스와 프로젝트에서 재사용할 운영 기준을 관리한다.

## Sections

- `convention/`
  프롬프트 브리프, 레퍼런스 관리, 결과물 정리 규칙을 둔다.
- `spec/`
  이미지 생성 파이프라인, 모델 선택, 후처리 기준을 둔다.
- `snippets/`
  반복적으로 쓰는 프롬프트 조각과 메타데이터 템플릿을 둔다.

## Rule

- 이미지 생성 요청은 브리프, 레퍼런스, 워크플로우, 산출물을 분리해 기록한다.
- 재현 가능한 결과가 필요한 작업은 모델, 시드, 크기, 프롬프트, 후처리 단계를 남긴다.
- 새 문서를 추가하면 이 인덱스를 함께 갱신한다.
