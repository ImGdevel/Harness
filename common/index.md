# Common Docs Index

공통 컨벤션과 공통 스펙 문서를 모아두는 루트 인덱스다.

## Sections

- `convention/`: 저장소 공통 규칙, 문서 규칙, 작업 규칙
- `spec/`: 하네스 구조, 워크플로우, 템플릿 스펙
- `templates/`: 공통 문서 템플릿

## Rule

- 새 문서를 추가하면 이 인덱스에 한 줄 요약과 위치를 기록한다.
- Git 작업 규칙은 `convention/` 아래 Git 관련 문서를 먼저 확인한다.
- 워크플로우 정의와 자동 실행 규칙은 `spec/` 아래 워크플로우 문서를 먼저 확인한다.
