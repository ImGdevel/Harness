# Templates Index

공통 문서 템플릿의 인덱스다.

## Templates

- `project-plan-template.md`
  실제 프로젝트 `plan/`에 저장하는 구현 계획 템플릿이다.
- `troubleshooting-template.md`
  실제 프로젝트 `troubleshooting/`에 저장하는 문제 해결 기록 템플릿이다.
- `github-pr-template.md`
  GitHub PR 본문을 작성할 때 사용하는 공통 템플릿이다.
- `github-issue-work-item-template.md`
  기능, 리팩터링, 문서, 운영 작업용 GitHub 이슈 템플릿이다.
- `github-issue-bug-template.md`
  버그, 장애, 회귀 상황용 GitHub 이슈 템플릿이다.
