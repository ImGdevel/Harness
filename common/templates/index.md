# Templates Index

공통 문서 템플릿의 인덱스다.

## Templates

- `project-plan-template.md`
  실제 프로젝트 `docs/plan/`에 저장하는 구현 계획 템플릿이다.
- `game-requirements-template.md`
  게임 프로젝트의 요구사항 정의와 MVP 범위를 정리하는 템플릿이다.
- `game-concept-art-brief-template.md`
  컨셉아트 작업을 위한 비주얼 브리프와 이미지 생성 입력 기준 템플릿이다.
- `game-design-plan-template.md`
  게임 시스템, 마일스톤, 기술 가정을 정리하는 설계 계획 템플릿이다.
- `troubleshooting-template.md`
  실제 프로젝트 `docs/troubleshooting/`에 저장하는 문제 해결 기록 템플릿이다.
- `github-pr-template.md`
  GitHub PR 본문을 작성할 때 사용하는 공통 템플릿이다.
- `github-issue-work-item-template.md`
  기능, 리팩터링, 문서, 운영 작업용 GitHub 이슈 템플릿이다.
- `github-issue-bug-template.md`
  버그, 장애, 회귀 상황용 GitHub 이슈 템플릿이다.
